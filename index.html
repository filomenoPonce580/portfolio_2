Intro

Life is complex and often very complicated, This is why I decided to become a software engineer. As an individual with a passion for learning and solving problems, I am committed to simplifying the human experience with the use of technology.

I have spent most of my working life in the restaurant industry where until recently, most of the technology used has been fairly outdated. Work

Table Tracker

A full stack reservation and table management system

I took on the challenge of designing and developing a comprehensive full-stack reservation and table management system. Leveraging my expertise in modern web technologies, I implemented a dynamic user interface using React, enabling seamless interactions for customers and restaurant staff.

On the backend, I harnessed the power of Express, a Node.js framework, to create a robust RESTful API that facilitated communication between the frontend and a PostgreSQL database. This API empowered users to effortlessly make reservations and retrieve relevant information. One of the project's highlights was the implementation of Create, Read, Update, and Delete (CRUD) functions, ensuring smooth data management. To handle database interactions efficiently, I utilized the Knex.js query builder, abstracting away raw SQL queries.

Git employed for version control.

Encryption/Decryption Application

An app for composing secret messages

The Decoder Ring application uses complex algorithms to execute the following encryption and decryption techniques:

  • Caesar Shift: Handles shifting of letters in the alphabet based on a given shift value
  • Polybius Square: Utilizes a 5x5 grid to encode and decode messages by mapping each letter to a unique pair of coordinates.
  • Substitution Cipher: Encrypts and decrypts messages by substituting letters with corresponding characters from a provided alphabet.
